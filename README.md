# LoXerium
An open-source server for [lox.exe](https://github.com/l-lacker/lox.exe) program. Customize it as you like.
## Supported features
- [ ] Basic functions
- [x] Customization features
- [ ] Lucky game
- [ ] Some hidden and useless features
# Build
**IMPORTANT: we currently dont support containers like Docker or Kubernetes**

**IMPORTANT No.2: DevelNext has experiencing problems when trying to fetch content from Github Pages. Use any other provider.**

### nginx (Standard config)
- git clone this repo to /var/www/html
- set up some libraries like app/walls and other stuff

*Don't forget to set up links in original lox.exe app!*
